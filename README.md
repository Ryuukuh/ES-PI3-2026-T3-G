# MesclaInvest - Aplicativo Mobile de Investimentos Simulados

Projeto desenvolvido para a disciplina de **Projeto Integrador 3 (PI3)** do curso de Engenharia de Software da **PUC Campinas** (2026).

## 🚀 Escopo do Projeto
O **MesclaInvest** é uma plataforma de simulação de investimentos focada no ecossistema de startups. O objetivo do aplicativo é permitir que usuários compreendam as dinâmicas de aportes financeiros e negociação de tokens de forma 100% simulada, utilizando dados realistas em um ambiente seguro.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** Flutter & Dart
- **Banco de Dados & Autenticação:** Firebase (Cloud Firestore & Firebase Auth)
- **Arquitetura:** Monorepo (Separação para Frontend e futuro Backend)
