# frontend — MesclaInvest

Este diretório contém o aplicativo Flutter do projeto MesclaInvest.

Para instruções completas de execução, tecnologias utilizadas e requisitos implementados,
consulte o [README principal na raiz do repositório](../README.md).

## Como executar

```powershell
cd frontend
flutter pub get
flutter run
```

> Execute sempre dentro da pasta `frontend/`, onde está o `pubspec.yaml`.
